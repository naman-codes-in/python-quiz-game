# Python Quiz Game 🎯

A fun quiz game built in Python with questions on Python, GK, and Anime!

## Features
- 5 questions on different topics
- Case insensitive answers
- Score at the end
- Hints for wrong answers

## How to Run
1. Clone the repository
2. Run `python main.py`

## Questions Topics
- Python Programming
- General Knowledge
- Anime 😄

## Scoring
- Sahi jawab = 1 point
- Galat jawab = hint milega
- End mein total score dikhega

## Concepts Used
- Object Oriented Programming (OOPs)
- Loops
- Lists
